## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import {
  Menu,
  MenuItem,
  MenuItemLabel,
} from '@/components/ui/menu';
```

```jsx
export default () => (
  <Menu>
    <MenuItem>
      <MenuItemLabel />
    </MenuItem>
  </Menu>
);
```
