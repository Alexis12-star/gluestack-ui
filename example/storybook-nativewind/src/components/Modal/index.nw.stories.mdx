## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import { Modal } from '@/components/ui/modal';
```

```jsx
export default () => (
  <Modal>
    <ModalBackdrop />
    <ModalContent>
      <ModalHeader>
        <ModalCloseButton></ModalCloseButton>
      </ModalHeader>
      <ModalBody />
      <ModalFooter />
    </ModalContent>
  </Modal>
);
```
